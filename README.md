# Super Mario Bros Remastered Android
This is an Android port of the [Super Mario Bros. Remaster by JHDev2006](https://github.com/JHDev2006/Super-Mario-Bros.-Remastered-Public).

## How to build
1. git clone -b 1.0.1 https://github.com/mircey/super-mario-bros.-remastered-android.git
2. launch Godot 4.5 beta 3
3. import /project.godot
4. Project -> Install Android build template
5. Project -> Export -> Android -> Use Gradle Build on
6. Export Project
